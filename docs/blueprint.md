# **App Name**: Grandeur Suites

## Core Features:

- Luxury Landing Page: Showcase a high-end hotel aesthetic, with image galleries and available services.
- User Authentication: Secure user login and JWT authentication using existing Spring Boot REST API.
- Dashboard: Display hotel occupancy statistics and booking charts using Recharts.
- Booking Management: Enable the ability to create, update, and delete hotel room bookings, including a calendar view and check-in/check-out functionality.
- Guest Database: Store guest information and booking history integrated with the Spring Boot REST API.
- Payment Processing: Record and manage hotel payments.
- Role Based Access Control: Implement role-based permissions for staff using JWT and the Spring Boot API to allow administrators control of user permissions.

## Style Guidelines:

- Primary color: Deep Navy Blue (#0B1F3A) for navigation, headings, and footer, to evoke trust, calm and luxury.
- Accent color: Gold/Champagne (#C9A24D) for buttons, icons, and highlights, to emphasize high quality.
- Background color: Soft Ivory (#F7F5EF) as the main background to impart a sense of gentle calm and quality.
- Headings font: 'Playfair' serif, to project luxury, high-end, and elegance.
- Body font: 'Inter' sans-serif, for clean and modern readability.
- Use gold minimal icons to enhance a premium feel.
- Implement clean spacing with balanced white space throughout the design.
- Subtle hover animations on buttons (slight scale and glow) and cards (lift with shadow increase) for an interactive experience.
